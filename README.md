##  Как устроен проект

Проект состоит из двух частей:

* **Generator** — создаёт CSV-файл с пони
* **Reporter** — анализирует данные и строит HTML-отчёт

Они общаются через общую папку `data/`.

---

##  Как запустить

```bash
./run.sh build_generator
./run.sh run_generator

./run.sh build_reporter
./run.sh run_reporter
```

---

##  Результат

После запуска появляется HTML-отчёт с:

* количеством пони в датасете
* статистикой по характеристикам
* распределением цветов и предпочтений
